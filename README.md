beast-analytics
===============

*Jupyter notebooks to process [the Beast](https://github.com/jjmrocha/beast) output files*


Requirements
------------
* Python 3.7
* Pipenv


Setup
-----
1. Create virtualenv
  ```
  pipenv --python 3.7
  ```

2. Install all dependencies
  ```
  pipenv install --dev
  ```
  
How-To Use
----------

Start jupyter lab 
```
pipenv run jupyter lab
```
